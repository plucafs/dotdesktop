# dotdesktop
A .desktop file CLI generator

## Build and run
```
# clone the repo
$ git clone https://github.com/plucafs/dotdesktop.git

# change the working directory to dotdesktop
$ cd dotdesktop

# build
$ nim c dotdesktop.nim

# run
$ ./dotdesktop
```
